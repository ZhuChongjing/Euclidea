<h1 align=center>Euclidea 解题参考</h1>

在我寻找攻略的时候看到了[OrangeTown](https://github.com/OrangeTown)这位老哥的[Euclidea攻略](https://notepad.readthedocs.io/en/latest/euclidea/euclidea.html)，里面有着所有关卡的解析。可是里面只有最终的图案，没有步骤，我只能盯着那张最终的图片反复猜测，非常不方便。于是，我决定要自己做一个网站，上面有详细的解题步骤，并且可以通过点击来查看每一步。我便开始构思如何搭建这个网站，在这个过程中，我也发现了[MathsFans](https://githubcom/MathsFans)的[Euclidea解题参考](https://mathsfans.github.io/Euclidea)（详见[MathsFans/Euclidea](https://github.com/MathsFans/Euclidea)），它的初始页面让我眼前一亮，我便果断使用并更改了一些样式，**在这里感谢MathsFans**；中间的页面我直接用了[Euclidea官网](https://euclidea.xyz)的源代码以及图片；最后是每个关卡的页面，我保留了前面的样式，并进行了重新布局与样式修改，将官网绘图时的按钮放到页面中，点击并显示该步骤。顺带一提，我本来是准备使用[ManimCE](https://github.com/manimCommunity/manim)来制作解题动画的，但随着关卡的难度增加，动画的制作难度也变得更高，所以我还是选择了直接在网页画图并截取页面。至此，整个网站的框架就完成了，就等着我去截取接下来每一关的步骤了。**（教程关不提供解析）**
